# Security Policy

## Supported Versions

As this project is a starter template, we only provide updates and security patches for the code currently residing on the `main` branch. 

## Reporting a Vulnerability

Security is our top priority. If you discover a security vulnerability within this template, please **do not** report it through a public GitHub issue.

Instead, please report it privately to the maintainer:
1. Contact directly at: **nafskrt@yahoo.com**
2. Or use GitHub's private vulnerability reporting feature in the "Security" tab of this repository.

Please include a detailed description of the vulnerability and the steps to reproduce it. We will try to address the issue as quickly as possible and provide a timeline for the fix.
